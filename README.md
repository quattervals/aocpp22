# Advent of Code 22 with C++

## Prerequisites
```
sudo apt install build-essential gcc pkg-config cmake
```

## Compile and Run
Configure: `cmake -B build`

Build: ` cmake --build build`
- Tests are run after every build

Run tests: `ctest --test-dir build`
